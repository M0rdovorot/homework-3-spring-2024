* Центр коммерции. Создание. Фид или сообщество. Ошибка при создании каталога с пустой полем "Ссылка на фид или сообщество"
* Центр коммерции. Создание. Вручную. При нажатии на кнопку "Скачать шаблон фида {выбранная категория фида}" начинается скачивание файла
* Центр коммерции. Создание. Вручную. Ошибка при загрузке файла фида формата, отличающегося от CSV, TXT, XML, YML
* Центр коммерции. Создание. Вручную. Ошибка при загрузке файла фида, превышающего 100 МБ
* Центр коммерции. Создание. Вручную. Ошибка при создании каталога без загруженного файла фида
* Центр коммерции. Создание. После создание каталога он добавляется в список каталогов
* ### Подраздел "Управление каталогом"
* Центр комерциии. Управление каталогом. Товары. При нажатии на кнопку "Рекламировать" отрывается форма создания кампании
* Центр коммерции. Управление каталогом. Товары. При нажатии на кнпоку "Добавить товары" открывается форма добавления товаров
* Центр коммерции. Управление каталогом. Товары. Добавление. При добавлении товаров с выбранным пунктом "ОБновить каталог" добавляются новые товары и/или обновляются существующие
* Центр коммерции. Управление каталогом. Товары. Добавление. При добвалении товаров с выбранным пунтком "Заменить каталог" удаляются все товара загруженные ранее
* Центр коммерции. Управление каталогом. Группы. При нажатии на кнопку "Создать группу" откырвается форма создания группы
* Центр коммерции. Управление каталогом. Группы. При создании группы с использование фильтров добавляется группа соответсвующих товаров в список групп
* Центр коммерции. Управление каталогом. Группы. При создании группы с товарами, выбранными вручную, добавляется группа соответсвующих товаров в список групп
* Центр коммерции. Управление каталогом. Диагностика. При нажатии на кнопку "Скачать полный отчет" начинается заргузка отчета
* Центр коммерции. Управление каталогом. События. При нажатии на кнопку "Подключить источник данных" открывается форма подключения источников данных
* Центр коммерции. Управление каталогом. События. Подключение источника данных. При нажатии на кнпоку "Добавить источник" открывается форма добавления источника
